# BestOf App - Documentation

## Module d'Authentification

L'application inclut un système d'authentification sécurisé.

### Identifiants de démonstration

Pour tester l'application, utilisez les identifiants suivants :

- **Administrateur** :
  - Nom d'utilisateur : `admin`
  - Mot de passe : `admin123`

- **Utilisateur** :
  - Nom d'utilisateur : `user`
  - Mot de passe : `user123`

### Fonctionnalités

- Connexion sécurisée
- Gestion des utilisateurs (CRUD)
- Protection des routes
- Déconnexion automatique

### Sécurité

Les mots de passe sont stockés localement dans le navigateur pour les besoins de démonstration.
En production, utilisez toujours un système d'authentification sécurisé avec hashage des mots de passe.
