pour que le site fonctionne correctement sur https://mfasolutions.github.io/bestof-app/, vous devez vérifier la configuration de GitHub Pages dans les paramètres de votre dépôt :

Allez sur https://github.com/MfaSolutions/bestof-app/settings/pages
Sous "Source", sélectionnez "Deploy from a branch"
Choisissez la branche gh-pages et le dossier / (root)
Sauvegardez les changements
